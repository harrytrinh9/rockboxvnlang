# rockbox VN lang
Vietnamese language for rockbox ver 3.15  
*Ngôn ngữ tiếng Việt cho Rom rockbox ver 3.15*  
>
>1. Chép file 'vietnam.lng' vào thư mục .rockbox\langs
>2. Vào Settings > General > Languages > vietnam
